# Overview
Flutter-проект (Lab 10) на Provider-архитектуре.

## Структура проекта
- lib/providers/ — ChangeNotifier-классы (Provider)
- lib/data/ — модели
- lib/network/ — сервисы
- lib/db/ — база данных и утилиты
- lib/ui/ — экраны

## Как собрать
- flutter pub get
- flutter run -t lib/main_provider.dart
